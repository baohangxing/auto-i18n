## 介绍

Auto-i18n 项目采用 monorepo 结构

- [cli](./packages/cli) 基于 core 包开发的命令行工具。

- [core](./packages/core) 提供了必要的支持的核心包

- [vsce](./packages/vsce) VS Code 插件
